# www2-pg-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存储，不针对任何特定域名或具体网站，仅作为通用页面存档使用。

## 项目简介

www2-pg-archive 是一个轻量的 HTML 页面集合仓库。您可以将需要长期保存或对外共享的 HTML 文件提交至此仓库，以便于版本管理和访问。

## 目录结构

```
www2-pg-archive/
├── index.html          # 仓库首页 / 导航页
├── pages/              # 归档的独立 HTML 页面
│   ├── example1.html
│   └── example2.html
└── assets/             # 页面引用的资源文件（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：仓库的入口页面，可提供页面列表或说明。
- `pages/`：存放所有归档的独立 HTML 文件，每个文件对应一个独立页面。
- `assets/`：存放页面所需的公共资源，可按需组织子目录。

## 页面归档说明

- 所有归档页面均为独立 HTML 文件，不依赖外部动态服务。
- 页面内容应保持完整，包括必要的样式和脚本（可通过 `assets/` 引用）。
- 归档文件命名建议使用简洁、有意义的英文或拼音，避免使用特殊字符。
- 若页面需要引用资源，请使用相对路径，确保在仓库内可正常访问。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面。
- 请确保新增页面不包含违规、敏感或侵犯他人权益的内容。
- 修改现有页面时，请尽量保持向后兼容，避免破坏已有引用。
- 仓库维护者会定期检查页面可用性，但不对页面内容的长期有效性做保证。

## 使用建议

您可以直接通过 GitHub Pages 或其他静态托管服务部署此仓库，以便在线浏览归档页面。部署后，每个 HTML 文件将拥有独立的访问地址。

## 许可

本仓库仅用于页面存档目的。页面内容版权归原作者所有，除非另有声明。
